# Evermos Virtual Internship Project - Golang Fiber

Proyek ini dikembangkan sebagai bagian dari program Rakamin x Evermos Virtual Internship.  
Aplikasi ini dibangun menggunakan bahasa pemrograman Go dengan framework Fiber, ORM GORM, dan database MySQL.  
Struktur proyek menggunakan pendekatan Clean Architecture agar lebih terorganisasi dan mudah dikembangkan.

## Teknologi yang Digunakan
- Go (Golang)
- Fiber v2 Framework
- GORM ORM
- MySQL Database
- JWT Authentication
- Postman / Thunder Client
- Clean Architecture

---

## Langkah Menjalankan Program

### 1. Clone Repository
```bash
git clone https://github.com/aulrizki/evermos-intern.git
cd evermos-intern

```

### 2. Konfigurasi Database
```sql
CREATE DATABASE evermosdb;

```

### 3. Install Dependencies
go mod tidy
go run main.go
